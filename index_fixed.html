🔑 Getting Your API Keys
Grace requires API keys to connect to AI services. Here's how to get them:
🚀 Groq API Key (For Chat AI)
Step 1: Go to console.groq.com
Step 2: Click "Sign Up" and create a free account (email or Google)
Step 3: Verify your email address
Step 4: Log in and go to "API Keys" section
Step 5: Click "Create API Key" and copy the key
Step 6: Paste it in the "Groq API Keys" field below
💡 Free tier: $5/month credit. Add multiple keys to extend usage.
🎨 Pollinations Key (For Images)
Step 1: Go to enter.pollinations.ai
Step 2: Click "Get API Key" or "Sign In"
Step 3: Create an account with email
Step 4: Go to Dashboard → API Keys
Step 5: Generate a new key and copy it
Step 6: Paste it in the "Pollinations Keys" field below
💡 Optional: Flux Schnell works without a key (free). A key unlocks higher quality models.
Use key to unlock klein-large (higher quality). Get key at enter.pollinations.ai - no key needed (flux free)
